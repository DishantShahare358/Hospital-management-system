# MediCore - Modern Hospital Management System

## Project Overview

MediCore is a comprehensive hospital management platform designed for healthcare professionals. It streamlines appointments, patient records, and medical operations.

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher) or Yarn

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:
   ```sh
   cd medi-core
   ```

3. Install dependencies:
   ```sh
   npm install
   # or
   yarn
   ```

4. Start the development server:
   ```sh
   npm run dev
   # or
   yarn dev
   ```

5. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Features

- Patient Management
- Appointment Scheduling
- Medical Records
- Staff Management
- Billing and Invoicing

## Building for Production

To create a production build:

```sh
npm run build
# or
yarn build
```

The build artifacts will be stored in the `dist/` directory.
